🚀 GitHub Pages Pe Deploy Karo

Pehle: Portfolio file download karo

Neeche "Download Portfolio" button se index.html file download karo.

Steps to go LIVE 🌐

1

GitHub account banao

Agar nahi hai toh github.com/signup pe free account banao.

2

New Repository banao

GitHub pe login karo → New button click karo → Repository name rakho: username.github.io (apna GitHub username likhna) → Public select karo → Create repository.

3

index.html upload karo

Repository ke andar jao → Add file → Upload files → Downloaded index.html drag karo → Commit changes click karo.

4

GitHub Pages enable karo

Repository → Settings → Pages → Source mein main branch select karo → Save.

5

🎉 Website LIVE!

2-3 minute baad tera portfolio live hoga: https://username.github.io — is URL ko kisi ko bhi share kar sakte ho!
